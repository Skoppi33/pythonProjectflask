    РЕЖИМЫ РАБОТЫ

    Для любого действия в 3D моделирование существует определенные режимы работы, которые включают в себя интерфейс и дополнительные функции. Режимы упрощают работу и позволяют разделить процесс создания модели поэтапно.

    Режимы blender:

    1. Object Mode


    2. Edit Mode


    3. Sculpt Mode


    4. Vertex Paint mode


    5. Weight Paint mode


    6. Texture Paint mode







    Object Mode (Режим Объекта):

    Назначение: Основной режим для управления целыми объектами.
    Функции:

    • Выделение, перемещение, вращение, масштабирование, дублирование и удаление объектов.
    • Применение модификаторов к объектам.
    • Переключение между объектами.
    Когда использовать: когда нужно работать с объектами как с единым целым, менять их положение в сцене, а не редактировать их геометрию.
    *Переключение в режим редактирования (Edit Mode) выполняется клавишей Tab клавиатуры. Повторное нажатие Tab вернёт в предыдущий режим. Также переходить в другой режим изменения объекта можно с помощью выпадающего списка в заголовке редактора.


    Edit Mode (Режим Редактирования):

    Назначение: режим для изменения геометрии (формы) объектов, например, мешей.
    Функции:

    • Выделение и перемещение, вращение, масштабирование отдельных вершин, рёбер и граней (полигонов).
    • Экструдирование, вырезание, добавление и удаление геометрии.
    • Различные инструменты для моделирования.
    Когда использовать: когда нужно изменить форму объекта, добавить или удалить полигоны, создать сложные модели из простых примитивов.
    *Для перехода в режим редактирования нужно нажать клавишу Tab или выбрать режим Edit Mode в заголовке окна 3D-вида.


    Sculpt Mode (Режим Скульптинга):

    Назначение: режим для моделирования “лепкой”, как если бы вы работали с глиной.
    Функции:

    • Различные кисти для добавления, сглаживания, выдавливания и других деформаций на поверхности модели.
    • Работа с динамической сеткой (Dyntopo) для добавления детализации.
    • Создание органических форм.
    Когда использовать: когда нужно создать сложные органические модели, персонажей, скульптуры, а также добавить детализацию к уже существующим моделям.


    Vertex Paint Mode (Режим Покраски Вершин):

    Назначение: : режим для покраски вершин объекта, позволяющий раскрасить объект без использования текстур (удобно для низкополигональных моделей).
    Функции:

    • Разные кисти для окраски вершин, их смешивания, размытия.
    • Раскраска в зависимости от освещения и других параметров.
    Когда использовать: когда нужно раскрасить объект, особенно простые низкополигональные модели, или добавить детализацию цвета без использования сложных текстур.


    Weight Paint Mode (Режим Рисования Весов):

    Назначение: режим для определения влияния костей скелета на деформацию меша при анимации.
    Функции:

    • Кисти для раскрашивания вершин, где красный цвет означает максимальное влияние, а синий — минимальное.
    • Настройка плавности влияния костей.
    Когда использовать: когда нужно анимировать модель, деформируя ее поверхность (например, сгибая руку).


    Texture Paint Mode (Режим Текстурного Рисования):

    Назначение: Режим для непосредственного рисования текстур на поверхности объекта.
    Функции:

    • Различные кисти для рисования, штампы, заливки.
    • Возможность использовать текстуры для кистей.
    • Рисование на UV-развертке модели.
    Когда использовать: когда нужно создать уникальные текстуры для объекта, нарисовав их прямо на модели.


    Ключевые моменты:

    • Зависимость: не все режимы доступны для всех типов объектов. Например, Edit Mode работает только с мешами.
    • Контекст: Режим работы определяет, какие инструменты доступны и как они себя ведут.
    • Освоение: чтобы по-настоящему освоить Blender, нужно понимать, какой режим использовать в какой ситуации.





    МОДИФИКАТОРЫ

    Важной частью 3D моделирвания является работа с модификаторами. Они придают необходимые свойства для объекта, к примеру: закругляют модель, добавляют тени и т.д.
    Всего в blendere 54 модификатора, которые делятся на 5 категорий, по принципу их работы

    Категории модификаторы:

    1. Edit


    2. Generate


    3. Deform


    4. Normals


    5. Physics
